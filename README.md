Installation
------------

You can install PlumDate using [Composer](http://getcomposer.org).

```shell
$ composer require plumphp/plum-date
```


Usage
-----

Please refer to the [Plum documentation](https://github.com/plumphp/plum/blob/master/docs/index.md) for more
information.
